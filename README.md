## 📂 Arrays

- One-dimensional arrays
- Multi-dimensional arrays
- Prefix Sum / Difference arrays
- Kadane's Algorithm
- Merge Intervals
- Subarray problems
- Matrix operations (rotation, transpose, etc.)

## 🧵 Strings

- String reversal, palindrome check
- String compression
- Anagram check
- Rabin-Karp, KMP, Z-algorithm
- String matching
- Longest Common Subsequence / Substring
- String hashing

## 🔁 Recursion & Backtracking

- Basic recursion patterns
- N-Queens
- Sudoku Solver
- Subset/Permutation/Combination generation
- Rat in a Maze
- Word Search

## 🧮 Mathematics

- GCD, LCM
- Prime numbers, Sieve of Eratosthenes
- Modular arithmetic
- Fast exponentiation
- Fibonacci series (DP + Matrix)
- Combinatorics (nCr, Pascal's Triangle)

## 📦 Searching & Sorting

- Binary Search (on answers too)
- Merge Sort / Quick Sort / Heap Sort
- Counting sort / Bucket sort
- Searching in Rotated Sorted Array
- K-th Element Problems

## 📊 Bit Manipulation

- Basic operators and shifts
- Checking power of 2
- XOR tricks
- Bitmasking
- Count set bits (Brian Kernighan’s algorithm)

## 🌲 Trees

- Binary Trees
- DFS/BFS Traversals
- Tree Diameter
- Lowest Common Ancestor (LCA)
- Tree serialization/deserialization

## 📈 Binary Search Trees (BST)

- BST insert/delete/search
- Inorder successor/predecessor
- Validate BST
- Convert BST to DLL / Balanced BST

## 📚 Heaps & Priority Queue

- Min Heap / Max Heap implementation
- Heapify & Build Heap
- Top K elements problems
- Median in a data stream

## 🔄 Stack & Queue

- Stack and Queue implementation
- Next Greater Element
- Min/Max Stack
- Infix to Postfix / Prefix
- LRU Cache
- Circular Queue / Deque

## 🧭 Graphs

- Representation (Adjacency Matrix/List)
- DFS/BFS
- Topological Sort
- Cycle Detection
- Dijkstra’s / Bellman-Ford / Floyd-Warshall
- Kruskal’s and Prim’s MST
- Union-Find (Disjoint Set)
- Bridges and Articulation Points

## 📑 Hashing & HashMaps

- Frequency count
- HashMap + Set problems
- Group Anagrams
- Longest Consecutive Sequence
- Subarrays with given sum

## 🔐 Sliding Window & Two Pointers

- Fixed and variable window problems
- Maximum sum subarray
- Longest substring with unique characters
- Container With Most Water

## 🧩 Greedy Algorithms

- Activity selection
- Fractional Knapsack
- Job Sequencing
- Huffman Encoding
- Interval Scheduling

## 🎲 Dynamic Programming (DP)

- 1D and 2D DP
- Memoization vs Tabulation
- Knapsack (0/1, Unbounded)
- LIS, LCS, Edit Distance
- DP on Trees
- DP on Bitmasks
- DP with State Compression

## 📏 Divide and Conquer

- Merge Sort, Quick Sort
- Binary Search
- Closest Pair of Points
- Majority Element

## 🧪 Miscellaneous

- Trie (Prefix Tree)
- Segment Tree / Fenwick Tree (BIT)
- Monotonic Stack/Queue
- Disjoint Set Union (DSU)
- Game Theory (Nim Game)
- Suffix Array / Suffix Tree
- Rabin Karp / Rolling Hash
